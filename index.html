📡 戶外直播設定教學

你應該已收到一個壓縮包（內含 install_noalbs.bat）和一份直播連線資料文字檔

這套系統在幹嘛？

讓手機畫面先傳到中繼站，再從中繼站傳到你的電腦 OBS。這樣手機訊號短暫斷掉，直播也不會馬上中斷。

📱你的手機IRL Pro / Moblin
→
🖥️中繼站幫你暫存畫面
→
🎬你的電腦 OBS抓畫面 + 推直播
→
🟣Twitch你的頻道
1 跑安裝工具（只要做一次）

解壓縮收到的壓縮包，雙擊 install_noalbs.bat，依照提示輸入五項資料：

工具詢問填什麼
① Twitch ID你的 Twitch 英文帳號
② Bot TokenNOALBS 會用你自己的帳號在聊天室發通知，不需要另外建機器人帳號。工具會幫你開網頁，確認已用你的帳號登入 Twitch 後，按 Connect，複製 oauth:xxxxxxxxxx 貼回來
③ 伺服器選擇輸入 1（flycat 伺服器）
④ OBS WebSocket 密碼OBS → 工具 → WebSocket 伺服器設定 裡自己設的密碼（見 Step 3）
⑤ 場景名稱直接按 Enter 使用預設（IRL / lowB / BRB）
完成後資料夾裡會有 啟動_NOALBS.bat 和 關閉_NOALBS.bat，之後每次直播用這兩個開關 NOALBS。
2 手機設定

URL 和 Stream ID 請整串貼上收到的直播連線資料文字檔裡的內容，不要自己修改。

少了推流帳號或密鑰的部分（例如只填 publish:你的Twitch ID）會連不上中繼站。
A

Android — IRL Pro

⚙ → Connections → New connection → 選 SRT → 填入文字檔裡的 URL 和 streamid → 回清單點選這組 → 主畫面按 Go Live。

建議畫質：1080p、30fps、Bitrate 3000–5000、Keyframe frequency 2。黑畫面請把編碼改成 H264。
B

iPhone — Moblin

⚙ → Streams → ＋ → 選 SRT(LA) → 填入文字檔裡的 URL 和 Stream ID → 開啟「Background streaming」→ 回主畫面按推流按鈕。

建議畫質：1080p、30fps、Bitrate 3000–5000。黑畫面請改 H264。
3 電腦 OBS 設定

需要三個場景（名稱要完全一樣）：

IRL
訊號正常
lowB
訊號變差
BRB
完全斷線
1

加入手機畫面（IRL 場景）

IRL 場景 → 來源 → ＋ → 媒體來源 → 取消「本機檔案」→ 輸入欄貼上直播連線資料文字檔裡的 OBS 媒體來源網址 → 輸入格式填 mpegts，重新連線延遲填 3 秒 → 確定。

黑畫面請把手機 App 的影像編碼改成 H264 / AVC。
2

推流到 Twitch

設定 → 串流 → 服務選 Twitch → 連結你的帳號。

3

開啟 WebSocket（讓 NOALBS 控制 OBS）

工具 → WebSocket 伺服器設定 → 勾選啟用 → 設定一組密碼（Step 1 安裝工具會問這個）→ 確定。

4 聊天室指令
指令效果誰能用
!start開始直播台主 / 管理員
!stop停止直播台主 / 管理員
!ss IRL手動切回正常畫面台主 / 管理員
!ss BRB手動切到等待畫面台主 / 管理員
!b查詢目前訊號強度所有觀眾
5 每次直播前的順序
1

開電腦 OBS

確認三個場景都在，IRL 場景有媒體來源。

2

雙擊「啟動_NOALBS.bat」

NOALBS 在背景執行，不會有視窗。稍等幾秒後在聊天室打 !b，有回應就代表連線成功。

3

手機開始推流

IRL Pro 或 Moblin 按推流按鈕，畫面開始傳到中繼站。

4

聊天室打 !start

NOALBS 自動幫你開始直播。直播結束後雙擊「關閉_NOALBS.bat」關閉。

訊號好壞會自動切換場景

訊號完全斷掉BRB
